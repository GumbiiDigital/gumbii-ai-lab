# Gumbii AI Lab

A public map of my local-first AI, agent, automation, and personal-intelligence experiments.

## Focus areas

- Local-first AI agents
- Voice-first and wearable AI workflows
- Browser/computer automation
- Local inference and GPU experiments
- Personal knowledge systems
- Public-data and OSINT interfaces

## Principles

- Prefer user-owned devices and local processing
- Keep private data private by default
- Build practical workflows over flashy demos
- Document setup, limits, and privacy behavior clearly

## Active tracks

### Local-first agents
Experiments around personal assistants that run on user-controlled hardware and talk to local LLMs.

- [even-g2-local-agent](https://github.com/GumbiiDigital/even-g2-local-agent) — local-first voice agent for Even Realities G2 glasses using Even Terminal Mode, Android, and Ollama.

### Wearable AI
Voice-first interfaces and glanceable HUD workflows for lightweight, hands-free computing.

### Browser/computer automation
Agent-controlled workflows for research, coding, and daily operations.

### Local inference
GPU and quantization experiments for running stronger models without depending on cloud APIs.

## Selected projects

- [`even-g2-local-agent`](https://github.com/GumbiiDigital/even-g2-local-agent) — local-first G2 voice agent template
- `spark-nvfp4-lab` — local GPU inference / NVFP4 experiments
- `browser-harness` — browser automation research
- `vibevoice` — open-source voice AI research
- `holaos` — agent-computer UX references
- `cookbook` — Gemma/local-model examples and notes

This repo is intentionally an index: individual projects may be experimental, reference forks, or active builds.
